# Chip-8 emulator implementation

A pet-project to learn more about emulation and computers.

## Requirements

- [go 1.17](https://go.dev)
- [go-sdl2](https://github.com/veandco/go-sdl2#requirements)

## Running

```
go run main.go <path/to/rom>
```

## Keymapping

| 1 | 2 | 3 | 4 |
| Q | W | E | R |
| A | S | D | F |
| Z | X | C | V |
